Portfolio

Minesweeper

Minesweeper was a very popular game back in the day which involved players opening cells which they think wouldn't contain a mine through hints showing the number of mines surrounding a particular cell.
This project was done by me using the Kotlin language and involved using Object oriented programming concepts like Classes, Constructors and Objects. I even learnt the singleton design pattern using companion is Kotlin. Lastly and most importantly I used the Flood-Fill algorithm which helped in providing hints to the players. Check out the project here.

Flashcards

Flashcards are tools that aid memorization such as historical events and their dates, Foreign words and words in your local language etc. This website does the same. The learner is presented with 9 cards the front side containing the name of a country and the back side containing the corresponding country's capital. When the learner hovers a particular card it flips over to show them the corresponding answer. Check out the website here and the project here.

Friendly Notifier

This was my first android app that I published on the Play Store. It is called Friendly notifier. It lets users set reminders and then fires an alarm when the reminder is due. The user can then delete reminders that have expired or delete reminders before they are due in case they have changed their mind. Additionally users can also update existing reminders such as change the reminder message or the time the reminder is due for. Download the app here.
